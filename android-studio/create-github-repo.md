# Android Studio から GitHub リポジトリを作成する

Flutter プロジェクトを Android Studio から
そのまま GitHub に上げる手順をまとめる。

---

## 手順（Android Studio からやるのが一番楽）

1. Android Studio でプロジェクトを開く
2. 上のメニューから  
   **VCS → Share Project on GitHub…** を選択
   なかったらGitからGithubを選んでShare Project on GitHub
